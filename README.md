# Dice Roll Game 
A terminal-based dice rolling game where you play multiple rounds against the computer and see who wins overall.

---

## 1. Description
- Language: Python
- Input: Number of dice + roll/quit command
- Output: Round result + final score summary
- Tracks: Wins for both user and computer

---

## 2. How It Works
1. User enters how many dice to use
2. User rolls or quits (enter "roll" or "quit")
3. Both user and computer roll the dice
4. Dice sums are compared (higher sum wins)
5. Scores tracked across all rounds
6. On quit, final scores and overall winner announced

---

## 3. Input / Output
| Scenario | User | Computer | Output |
|---|---|---|---|
| Win | 5 + 4 = 9 | 2 + 3 = 5 | "You won this round." |
| Lose | 1 + 2 = 3 | 4 + 5 = 9 | "Computer won this round." |
| Draw | 3 + 3 = 6 | 2 + 4 = 6 | "Match draw." |
| Quit | quit | - | Final score summary |
